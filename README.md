# Ecto project with reference to book "Programming Ecto - The Pragmatic programmers"

## Setup

To setup dependencies: 
```
  mix deps.get
```

To setup the project: 
```
  mix ecto.setup
```

Do run the following to see few insights:
```
  mix app.stats
```
